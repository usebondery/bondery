# 🟣 Welcome

## What is Bondery?

Bondery is an open-source personal relationship manager (PRM). It helps you stay meaningfully connected with the people in your life: friends, family, colleagues, or anyone you want to keep in touch with.

## What can you do with Bondery?

### For everyone

* **Manage contacts** — Keep a rich database of the people in your network, with notes, social links, phone numbers, and more
* **Log interactions** — Record calls, meetings, coffees, and messages so you have a timeline of every relationship
* **Set reminders** — Add birthdays, anniversaries, and other important dates and get email notifications before they arrive
* **Import from LinkedIn & Instagram** — Use the browser extension to save contacts straight from social networks

### For developers

* **REST API** — Access and manage your data programmatically via the [API](api/api-reference/README.md)
* **Self-host** — Run the full stack yourself; Bondery is fully open-source under AGPL-3.0
* **Contribute** — Extend or improve the platform; see the [Contributing guide](contributing/README.md)

## Current stage

Bondery is currently in beta. You may encounter occasional bugs, and some features may change as we improve the platform. If you run into something, please [open an issue](https://github.com/usebondery/bondery/issues).
