# 🟣 Welcome

## What is Bondery?

Bondery is an open-source personal relationship manager (PRM). It helps you organize and maintain your personal and professional contacts in one place. The goal is simple: make it easier to build and keep meaningful relationships over time.

## What you can do with Bondery?

* Manage contacts in a central place
* Access your data through the API
* Use Bondery in automated workflows and integrations
* Keep product and API behavior aligned through shared types and docs

## Current stage

Bondery is currently in beta. You may encounter occasional bugs, and some features can change as we improve the platform.
