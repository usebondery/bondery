# Mobile app

Bondery is available as a native iOS and Android app built for quick contact access on the go, with offline support for your core contact data.

{% hint style="info" %}
The mobile app is in active development. Some features available on the web are not on mobile yet — see [What's available](#whats-available) below.
{% endhint %}

## Sign in

Use the same account as the webapp. Sign in with **GitHub** or **LinkedIn** — Bondery does not use email/password login.

## What's available

| Feature | Mobile |
|---|---|
| Contacts — browse, search, sort | Yes |
| Contact detail — phones, emails, socials, addresses, notes | Yes |
| Groups — view, assign, manage members | Yes |
| Tags — assign and manage | Yes |
| Important dates | Yes |
| Myself profile | Yes |
| Offline access — view and edit contacts, groups, tags | Yes |
| Share contact (vCard, email) | Yes |
| Swipe actions (call, message, email) | Yes — configurable in Settings |
| Theme and language (EN / CS) | Yes |
| Map view | Web only |
| Interactions timeline | Web only |
| AI Assistant (chat) | Web only |
| Chrome extension capture | Web/extension only |
| Bulk LinkedIn / Instagram ZIP import | Web only |

Edits you make offline sync automatically when you reconnect. A sync status banner shows pending changes and conflicts.

## Navigation

The app uses a floating tab bar on main screens:

- **Contacts** — your contact list with alphabet scrollbar and multi-select
- **Add** — create a new contact
- **Settings** — preferences, tags, group sort order, account

Deeper screens (contact detail, group view, notes editor) hide the tab bar until you go back.

## Getting the app

Distribution is rolling out through TestFlight and Play Store internal testing. If you self-host, build from source — see [Local development setup](../contributing/local-setup.md#7-mobile-application-appsmobile).
