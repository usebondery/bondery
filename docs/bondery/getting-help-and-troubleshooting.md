# 🤔 Getting help and troubleshooting

If something is not working as expected, please let us know. We'll be happy to help. You have three options:

1. Contact the Bondery team
2. Open an issue on GitHub
3. Open a PR on GitHub

### 1. Contact the Bondery team

You can contact us by email at `{{space.vars.support_email}}` . You can also find other contact options on our website:

### 2. Open an issue on GitHub
