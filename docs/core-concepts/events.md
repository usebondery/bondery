# Events

Events are logged interactions between you and one or more contacts. They give you a timeline of how your relationships have developed over time.

## Event types

| Type                  | Description                                             |
| --------------------- | ------------------------------------------------------- |
| Call                  | A phone or video call                                   |
| Coffee                | An informal meetup                                      |
| Email                 | An email exchange                                       |
| Meal                  | Lunch, dinner, or any shared meal                       |
| Meeting               | A formal or scheduled meeting                           |
| Networking event      | A conference, meetup, or similar event                  |
| Note                  | A general note, not tied to a specific interaction type |
| Party/Social          | A social gathering                                      |
| Text/Messaging        | A chat or messaging exchange                            |
| Competition/Hackathon | A shared competition or hackathon                       |
| Other                 | Anything that does not fit the above                    |
| Custom                | A user-defined label                                    |

## What an event holds

| Field        | Description                                           |
| ------------ | ----------------------------------------------------- |
| Type         | One of the event types above                          |
| Title        | An optional short label for the event                 |
| Description  | Free-form notes about what happened                   |
| Date         | When the interaction took place                       |
| Participants | One or more contacts who were part of the interaction |

## Effect on contacts

Logging an event with a contact updates the `lastInteraction` field on that contact. This lets you see at a glance who you haven't been in touch with recently.

## Related concepts

* [People](people.md) = the participants of an event
* Reminders = proactive notifications for upcoming important dates
