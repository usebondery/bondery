# Tags

Tags are color-coded labels you can attach to any number of contacts. They let you slice and filter your network in a lightweight way — think "investors", "economics", or "follow-up-q2".

## Tags vs. Groups

Both tags and groups can be applied to many contacts, but they serve different purposes.

**Tags** answer the question *"what kind of person is this?"* They are flat labels for categorization and filtering. A tag has no history of its own — it is just a marker that makes contacts easier to find and segment.

**Groups** answer the question *"what context do we share?"* A group represents a real-world context: a team, an alumni cohort, a social circle, a community. Groups give a shared identity to a set of contacts so you can navigate and reference them together.

**Rule of thumb:** if you would search for "all contacts tagged X", use a tag. If you would say "I had a meeting with this group of people", use a group.


## Creating and managing tags

Go to **Settings → Tags** to:

- Create a new tag
- Rename a tag
- Delete one or more tags (this removes them from all contacts)

## Assigning tags to contacts

Open any contact's page and look for the **Tags** section. From there you can:

- Assign one or more existing tags
- Remove a tag from that contact

A single contact can have any number of tags.

## Filtering by tag

The contacts list supports filtering by tag. Select a tag in the filter panel and only contacts with that tag will be shown. You can combine tag filters with the search box and other filters.
