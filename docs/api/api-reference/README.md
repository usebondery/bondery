# API reference

## Introduction

In addition to the traditional dashboard, Bondery also provides an API. This lets you use Bondery as a central "contact book" for your workflows, from AI agents to automated uploads using API in your own apps.

{% hint style="warning" %}
Bondery is currently in beta, so you may see occasional bugs, and some features may change without notice. However, we keep the documentation up to date so it always reflects the current product state.
{% endhint %}

## Base URL

All API requests should be made to the following base URL:

```
https://api.usebondery.com/
```

If you're selfhosting, change the path to your own API server URL.

## Issues and contributing

If you run into any issues, please contact us or open an issue on GitHub. Read more here: [getting-help-and-troubleshooting.md](../../bondery/getting-help-and-troubleshooting.md "mention").

We're also open to contributions, so feel free to open pull requests!
