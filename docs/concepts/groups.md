# Groups

Groups let you organize people into labeled categories. You can create as many groups as you need and assign any person to one or more of them.

## What a group holds

| Field | Description                                                  |
| ----- | ------------------------------------------------------------ |
| Label | The display name of the group (e.g. "Close friends", "Work") |
| Emoji | An optional emoji to visually identify the group             |
| Color | An optional color used in the UI                             |

## Assigning contacts to groups

People and groups have a many-to-many relationship. A single contact can belong to multiple groups, and a group can contain any number of contacts.
