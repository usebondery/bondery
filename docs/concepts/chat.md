# AI Assistant

The AI Assistant lets you interact with your Bondery data using natural language. Ask questions, search contacts, log interactions, and more — all through a conversational interface.

## Capabilities

The assistant can:

| Capability             | Description                                                        |
| ---------------------- | ------------------------------------------------------------------ |
| **Search contacts**    | Find people by name, location, language, tags, or any other field  |
| **View contact details** | Pull up full profiles including phones, emails, and notes        |
| **Log interactions**   | Record that you met, called, or messaged someone                   |
| **Query interactions** | Ask about recent interactions or look up past meetings             |

## Sample prompts

Here are some things you can try:

- "Who have I not talked to in a while?"
- "Show me contacts in New York"
- "I had coffee with Blake today"
- "Who speaks Spanish?"
- "What interactions did I log this week?"
- "Show me details for Jane Doe"

## How it works

1. You type a message in the chat input.
2. The assistant interprets your request and uses built-in tools to query your Bondery data.
3. Results are displayed as interactive cards you can click to navigate to the full record.

The assistant can chain multiple tool calls in a single response — for example, searching for a contact and then fetching their details.

## Data privacy

- The AI Assistant only has access to **your own data**, enforced by the same Row Level Security policies as the rest of Bondery.
- Your messages and data are sent to the AI provider (Anthropic) for processing but are **not used for training**.
