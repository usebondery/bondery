# Map view

The map view plots every contact that has a location on an interactive world map. It is a good way to see where your network is concentrated, plan visits, or find contacts in a specific city before you travel.

## Prerequisites

A contact appears on the map only if their **location** field has been geocoded — that is, resolved to a latitude and longitude. When you type a location on a contact's page, Bondery suggests places as you type and stores the coordinates automatically. Contacts with a free-text location that was never geocoded will not appear.

## Navigating the map

### Location search

Use the **search bar** at the top of the map to fly to any city, region, or address. Select a suggestion and the map pans and zooms to that point. Contacts in the nearby area will be highlighted in the panel below.

### Viewport table

Below the map there is a contact table that lists only the contacts currently visible in the viewport. As you pan or zoom, the table updates automatically. You can:

- **Search** within visible contacts using the table's search box
- **Sort** by first name, last name, or last interaction date (ascending or descending)
- **Show/hide columns** — Name, Headline, Location, Last Interaction, Social Media
- **Select** one or multiple contacts using the checkboxes (shift-click to select a range)

## Markers

Each contact with valid coordinates is shown as a marker on the map. Clicking a marker opens a small popover with the contact's name and a link to their full profile page.
